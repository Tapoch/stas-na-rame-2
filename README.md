# Стас на раме 2

Продолжение легендарной игры. Стек: vue 2.6, vue-router, vuex, firebase, tailwind. Для подключения к firebase необходим
.env.local файл с соответствующими данными.

## Установка

```
npm install
```

### Локальная разработка с hot-reload

```
npm run serve
```

### Сборка в продакшн

```
npm run build
```
